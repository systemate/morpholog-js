Morpholog-JS
============

Mopholog-JS is Javascript library for russian language rules:
* Decline (person names, phrases, geographic names);
* Numeric to numerals convension;
* Text tokenization;
* Word stemming;
* Sex determination;
* Plural/singular form detection/convension;
* Fwords detection;
